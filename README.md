# 🌿 Greenden - Plant E-Commerce Website

A modern and responsive plant e-commerce website built using **HTML5** and **Tailwind CSS**. Greenden provides an elegant user interface for showcasing plant collections, customer testimonials, and contact information while ensuring a seamless experience across all devices.

## 🚀 Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and clean user interface
* Product showcase section
* Featured plant collections
* Customer testimonials
* Contact section
* Fast-loading and user-friendly layout
* Built with Tailwind CSS utility-first framework

## 🛠️ Technologies Used

* HTML5
* Tailwind CSS

## 📂 Project Structure

```text
Greenden/
│
├── index.html
├── products.html
├── contact.html
├── images/
└── README.md
```

## 🎯 Purpose

This project was developed to practice and demonstrate responsive web design skills using HTML and Tailwind CSS. It focuses on creating a visually appealing and user-friendly interface for a plant shopping platform.

## 💻 Getting Started

1. Clone the repository:

```bash
git clone https://github.com/gomathi07-cyber/greenden.git
```

2. Navigate to the project folder:

```bash
cd greenden
```

3. Open `index.html` in your browser.
